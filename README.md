# Project Nugget SDK
A SDK for creating mods and maps in Project Nugget.

**Project Nugget is not released yet, and this SDK will not be downloadable until then. Join our Discord at the bottom for information on release dates.**

## Requirements
This is the requirements for the modding SDK, so you know what you need before we get started!

- A computer
- Unity 2022.3.45f1 with Android modules downloaded + OpenJDK

The modding SDK has only been tested on Windows 11, other operating systems may not work as expected.

## Setup
Ensure you have all previously stated requirements, then you can read our [wiki](https://github.com/Nugget-Analytics/ProjectNuggetSDK/wiki) to learn how to actually create mods & maps with the Project Nugget Modding SDK.
